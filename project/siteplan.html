RH / DSGN — Site Plan

I build the engine (Dev) and I drive the traffic (Marketing).

Site Name

RH / DSGN

This name reflects the purpose of the website: showcasing my skills, works and qualifications as a freelance Web Developer and community manager towards building my personal brand.

Site Purpose

The website serves as a dual-portfolio and service hub. It needs to demystify the technical side of development while showcasing the ROI (Return on Investment) of marketing.

Scenarios

  • The Small Business Owner: "I need a website, but I don't know how to get people to visit it. Can you handle the tech AND the Instagram ads so I don't have to hire two agencies?"
  • The Creative Agency Lead: "I have a great design, but I need a developer who understands aesthetics and won't break the layout. Can you code pixel-perfect designs"
  • The Startup Founder: "We need a raw, fast MVP (Minimum Viable Product). Can you build it quickly without it looking generic?"

Color Scheme

Background Color: #F0F0F0 (Raw Concrete) — Main canvas. Not pure white (too clean), gives a raw paper feel.

Foreground Color: #111111 (Ink Black) — Text, heavy 3px borders, structural grids.

Primary Color: #D92B2B (Bauhaus Red) — CTAs, hover states, urgent alerts.

Secondary Color: #002FA7 (International Klein Blue) — Tech elements, code snippets, links.

Accent Color: #FFD700 (Warning Yellow) — Highlights, badges, energetic geometric shapes.

I plan to use these colors for my headings, section borders, and visual highlights.

Typography

  • Heading Font: Space Grotesk — used for all page headings (h1, h2, h3).
  • Body Font: Public Sans — used for paragraphs and general text.
